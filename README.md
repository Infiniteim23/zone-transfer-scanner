## 📋 Overview

Zone Transfer Scanner detects DNS zone transfer (AXFR) vulnerabilities.

### ✨ Features

- 🚀 **Fast**: Async scanning
- 🎯 **Multi-Domain**: Scan single or multiple domains
- 🔧 **Flexible**: Custom ports, nameservers
- 📊 **Multiple Outputs**: JSON, CSV, TXT
